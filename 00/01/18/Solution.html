Approach 1: Dynamic Programming

Intuition

If we have the a row of Pascal's triangle, we can easily compute the next row by each pair of adjacent values.

Algorithm

  • Time complexity : O ( n u m R o w s 2 ) O(numRows^2) O(numRows2)

    Although updating each value of triangle happens in constant time, it is performed O ( n u m R o w s 2 ) O(numRows^2) O(numRows2) times. To see why, consider how many overall loop iterations there are. The outer loop obviously runs n u m R o w s numRows numRows times, but for each iteration of the outer loop, the inner loop runs r o w N u m rowNum rowNum times.
